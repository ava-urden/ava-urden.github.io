Integritetspolicy – Ava Growth Radar

Denna policy beskriver hur Ava Growth Radar (trendspaning/dashboards) hanterar data. Tjänsten används för intern analys och samlar inte in personuppgifter om besökare.

Senast uppdaterad: 22 februari 2026.

Data som hanteras

  • Trenddata från Pinterest API (endast läsning).
  • Valfri access token som sparas lokalt i webbläsaren om du själv anger den.
  • Grundläggande webbanalys via Google Tag Manager (sidvisningar/aggregat).

Användning

  • Token används endast för att hämta trenddata och lagras lokalt i din webbläsare.
  • Ingen data skickas till egna servrar för lagring.
  • Trenddata visas i gränssnittet och sparas inte på server.

Delning & lagring

  • Vi säljer inte data och delar inte personuppgifter.
  • Data delas endast med leverantörer som krävs för funktion (Pinterest API, Google).
  • Du kan rensa lokal data genom att ta bort token i appen eller rensa webbläsarens lagring.

English summary

Ava Growth Radar is an internal trend dashboard. It does not collect personal data. If you provide a Pinterest access token, it is stored locally in your browser and used only to fetch trend data from the Pinterest API. We do not store data on our own servers. Basic analytics may be collected through Google Tag Manager.
